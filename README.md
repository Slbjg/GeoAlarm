## Описание проекта

Умный гео-будильник – это мобильное приложение для Android. Приложение отслеживает местоположение пользователя и срабатывает (звук + вибрация + уведомление) при входе в заданную GPS-зону или при выходе из неё.

Приложение разработано на языке Kotlin с использованием Android SDK. В основе работы лежат компоненты LocationManager для получения геопозиции, а также MediaPlayer и Vibrator для сигнала.

Приложение позволяет пользователю:
- задать координаты центра зоны (широта и долгота);
- установить радиус зоны в метрах;
- выбрать тип срабатывания: при входе в зону или при выходе из зоны;
- получать звуковой сигнал, вибрацию и push-уведомление при срабатывании;
- видеть текущее местоположение на экране.

## Основной сценарий работы приложения

1. Пользователь запускает приложение и вводит координаты центра зоны (широту и долготу).
2. Задаёт радиус зоны (по умолчанию 100 метров).
3. Выбирает тип срабатывания: «При входе в зону» или «При выходе из зоны».
4. Нажимает кнопку «Установить гео-будильник».
5. Приложение запрашивает разрешение на определение местоположения (если не выдано) и начинает слежение.
6. Когда пользователь входит в зону (или выходит из неё, в зависимости от выбора):
   - воспроизводится звуковой сигнал (6 секунд);
   - включается вибрация (2 секунды);
   - появляется всплывающее уведомление;
   - отправляется push-уведомление в шторку.
7. Слежение автоматически останавливается после срабатывания.

## Реализованный функционал

| Категория | Реализованные возможности |
|-----------|---------------------------|
| Определение местоположения | GPS-провайдер, обновление каждые 2 секунды или 1 метр |
| Вычисление расстояния | Location.distanceTo() |
| Типы срабатывания | Вход в зону / Выход из зоны |
| Звуковой сигнал | MediaPlayer, системная мелодия будильника |
| Вибрация | Vibrator (2 секунды) |
| Уведомления | NotificationCompat, канал для Android 8+ |
| Отображение статуса | Текущие координаты, расстояние до зоны, статус |
| Разрешения | ACCESS_FINE_LOCATION, ACCESS_COARSE_LOCATION, VIBRATE |

```
GeoAlarm/                            # Корневая папка проекта
└── app/                             # Модуль приложения
    └── src/
        └── main/
            ├── java/com/example/geoalarm/
            │   └── MainActivity.kt          # Главная активность (GPS, логика, звук, уведомления)
            ├── res/
            │   ├── layout/
            │   │   └── activity_main.xml    # Интерфейс (поля ввода, кнопки, статус)
            │   └── values/
            │       ├── themes.xml           # Тема приложения (зелёная цветовая схема)
            │       ├── themes.xml (night)   # Тёмная тема
            │       ├── colors.xml           # Цвета приложения
            │       └── strings.xml          # Название приложения
            ├── AndroidManifest.xml           # Манифест (разрешения, активность)
            └── build.gradle.kts              # Сборочный файл модуля
```

### Назначение файлов

| Файл | Назначение |
|------|-------------|
| `MainActivity.kt` | Управление интерфейсом, запрос разрешений, получение GPS-координат, проверка входа/выхода из зоны, воспроизведение звука и вибрации |
| `activity_main.xml` | Разметка экрана: поля ввода координат и радиуса, Spinner для выбора типа срабатывания, кнопки, TextView статуса |
| `themes.xml` | Тема приложения (основной цвет #4CAF50 – зелёный) |
| `strings.xml` | Строковые константы (название приложения) |
| `AndroidManifest.xml` | Регистрация активности, запрос разрешений на геолокацию и вибрацию |
| `build.gradle.kts` | Настройки сборки: minSdk 24, targetSdk 36, зависимости |

## Установка и запуск

### 1. Копируйте репозиторий

```bash
git clone https://github.com/S1bjg/GeoAlarm.git
```

### 2. Откройте проект в Android Studio

`File → Open` → выберите папку проекта → дождитесь синхронизации Gradle

### 3. Подготовьте устройство

- Включите **GPS** на телефоне
- Включите **«Режим разработчика»** и **«Отладку по USB»** (если запускаете с компьютера)
- Разрешите приложению доступ к геолокации при первом запуске

### 4. Запустите приложение

Нажмите зелёную кнопку **Run ▶️**

> Важно: Для работы GPS необходим реальный телефон. Эмулятор может не давать точных координат.

| Шаг | Действие |
|-----|----------|
